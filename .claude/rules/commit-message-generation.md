# コミットメッセージ生成の指示

## Priority Markers

> 優先度の定義については [CLAUDE.md](../../CLAUDE.md#priority-markers) を参照。

## 必須フォーマット

`<type>(<scope>): <subject>`

### type

- `feat` / `fix` / `update` / `upgrade` / `bump` / `docs` / `style` / `refactor` / `test` / `chore`

### scope

- 英語・小文字

### subject

- 英語・命令形・50文字以内
- 先頭は小文字
- 末尾にピリオド不要
